# Anti Gravity

Anti Gravity is a flagship enterprise Shopify storefront specification.

The repository itself is the source of truth.

All numbered markdown files build upon previous documents.

When using AI:

1. Read every numbered markdown file before writing anything.
2. Determine the highest numbered completed document.
3. Continue from the next logical document.
4. Never recreate existing files.
5. Never overwrite previous specifications.
6. Maintain identical writing style and engineering philosophy.
7. Assume the project will eventually contain approximately 100 engineering specification documents.

The AI should treat this repository as internal engineering documentation rather than a collection of prompts.
